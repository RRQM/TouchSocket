# TouchSocket.Hosting

## 概述
TouchSocket.Hosting 是 TouchSocket 基于通用主机的扩展项目，提供了 Tcp、Udp、NamedPipe、Dmtp、SerialPort 等服务的支持。借助该项目，开发者能够更便捷地将这些网络服务集成到基于通用主机的应用程序中。

## 支持的目标框架
- net462
- net472
- netstandard2.0
- netstandard2.1
- net6.0
- net9.0
- net8.0
