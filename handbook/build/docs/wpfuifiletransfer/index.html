WPF界面、文件传输项目

定制方 凯**有限公司 说明 应该公司要求，开发以WPF为技术框架的桌面项目管理应用，其界面类似QQ（此处因为涉及UI版权，不展示实际效果）。并且以RRQMSocket为通信基础，解决项目文件的上传和下载。实际的解决了在2G网，甚至更差的网络环境下的大文件传输。

技术点

  • 整体界面：圆角窗体、窗体阴影、最小化嵌入、拖拽效果等。
  • 项目树视图：样式、搜索、定向搜索、搜索项定位、树视图连线。
  • 数据同步：设置配置数据、项目文件数据等。
