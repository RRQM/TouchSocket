## 一、3DES
**Nuget Package：**[TouchSocket](https://www.nuget.org/packages/TouchSocket/)
【加密】
```csharp
var dataLocked = DataSecurity.EncryptDES(data, "12345678");//加密口令，长度为8。
```
【解密】
```csharp
var newData = DataSecurity.DecryptDES(dataLocked, "12345678");//解密口令，和加密一致。
```
