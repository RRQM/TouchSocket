## 1.3.1 示例架构

源码仓库中的 `samples` 示例采用多层分层设计，主要设计是 `自动抽象工厂` 和 `DDD 领域驱动设计` 相结合。

`TouchSocket` 示例项目结构如下：

- `TouchSocket`：框架核心层
- `TouchSocket.Application`：业务应用层（业务代码主要编写层）
- `TouchSocket.Core`：核心层（实体，仓储，其他核心代码）
- `TouchSocket.Database.Migrations`：EFCore 架构迁移文件层
- `TouchSocket.EntityFramework.Core`：EF Core 配置层
- `TouchSocket.Web.Core`：Web 核心层（存放 Web 公共代码，如 过滤器、中间件、Web Helpers 等）
- `TouchSocket.Web.Entry`：Web 入口层/启动层

示例地址：[https://gitee.com/rrqm_home/TouchSocket/tree/v4/samples](https://gitee.com/rrqm_home/TouchSocket/tree/v4/samples)
