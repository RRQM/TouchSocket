## 9.1 动态订阅消息

```cs showLineNumbers 
MessageCenter.Subscribe("messageId", async (ctx) => {
    Console.WriteLine("我是动态的");
    await Task.CompletedTask;
});
```

## 9.2 发送消息

```cs showLineNumbers 
await MessageCenter.PublishAsync("messageId", new {});

// 诸多重载
```

## 9.3 取消订阅

```cs showLineNumbers 
MessageCenter.Unsubscribe("messageId");
```
